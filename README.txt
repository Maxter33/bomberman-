DESCRIÇÃO DO TRABALHO


-> Esse trabalho foi feito em C e utilizando a biblioteca grafica Raylib, esse trabalho contem todos os requisitos obrigatorios e tambem possuem um sistema de audio funcional. NOTE-SE que os audios ficam fazendo o load durante alguns processos do programa sendo possivel ver no prompt, é possivel que ao iniciar o programa ele feche, e necessario inicia-lo novamente caso isso aconteça. TODO o trabalho foi feito por mim (João Victor Ribeiro dos Santos).

OBS: É necessario colocar todos os mapas no arquivo maps, e em formato "mapa<numero>.txt".
